About MLC Dataset Catalog

The MLC Dataset catalogue contains descriptions of 89 MLC datasets. Each dataset is annotated with a description of the problem the dataset is associated with and the different transformations performed. All dataset descriptions are enhanced with semantic annotations (metadata) based on classes/terms from ontologies and controlled vocabularies. The semantic annotations can be categorized into two groups: (1) Annotations of the dataset provenance information and (2) Annotations that capture the relevant machine learning characteristics of the datasets.
For a semantic description of provenance information, Schema.org vocabulary is used. Schema.org is a shared collection of schemas widely used for providing structured data on the Web. Specifically for annotation of the MLC datasets, the Dataset schema is used. It provides a list of properties that can be used for annotation, such as name, description, identifier, citation, and license, among others.
From an ML perspective, there are different types of annotations that are relevant and beneficial, such as annotation of dataset specification, learning task, datatypes, and meta-features. To enable annotation of ML-specific information, ontological concepts from two external ontologies are combined, i.e., the ontology of core data mining entities (OntoDM-core), and the ontology of datatypes (OntoDT). In order to further characterize the MLC datasets, MLC-specific meta-features are made available. Meta-features describe the MLC datasets from a task level, as they encode the knowledge for the task properties. As such, meta-features can be used as a base of an empirical study for the properties of the learning task or to guide practitioners to understand better the common challenges for solving their problem. MLC meta-features are grouped in several groups: (1) Dataset specific meta-features, such as the number of attributes, data instances, and labels; (2) Attribute specific meta-features, containing statistical and information-theoretic properties of the attributes; (3) Label specific meta-features that describe the distribution of the label space, as well as the relationship between the labels. All meta-features are calculated using two Java-based libraries, i.e., MLDA and MULAN, that implement the meta-features.
All semantic annotations are Open and published under the https://creativecommons.org/licenses/by/4.0 license. For each dataset in the catalogue the semantic annotations can be downloaded in RDF (subject, predicate, object) triples. The publicly available datasets are also available for download in ARFF format. The calculated MLC meta-features that appear as RDF triples in the semantic annotations are also available for download in JSON format.
